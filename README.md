## Coursework Project – UN SDG 3: Good Health and Well-being

This project is a coursework-based website developed in alignment with **UN Sustainable Development Goal 3 (Good Health and Well-being)**. It focuses on promoting health awareness and encouraging community engagement through an accessible and informative web platform.

---

## Project Overview

The website is designed to educate users on important health topics, provide awareness on well-being practices, and motivate individuals to adopt healthier lifestyles. It also aims to foster community involvement by sharing useful resources and information.

---

## Technologies Used

- HTML5  
- CSS3  
- JavaScript  

---



The goal of this project is to contribute to global health awareness by leveraging web technologies to educate and engage users in maintaining a healthier lifestyle.
